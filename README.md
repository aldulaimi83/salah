# Salah Kids

A static multilingual website for Muslim children abroad to learn the basics of wudu and salah.

## Features

- Kid-friendly wudu guide
- Beginner salah guide
- English, Arabic, Spanish, French, Urdu, and Turkish interface
- Right-to-left layout for Arabic and Urdu
- Daily local practice tracker
- GitHub Pages-ready static files

## Files

- `index.html` - page structure
- `styles.css` - visual design and responsive layout
- `script.js` - translations, lesson rendering, and tracker
